# 数学启蒙 App 市场调研报告

## 1. 调研目标

调研市场上面向 5-6 岁儿童的数学启蒙 App，重点关注加减法运算类产品的功能设计、玩法机制、商业模式与用户反馈，为自研 iPad 端产品提供参考。

---

## 2. 国际市场主流产品

### 2.1 Khan Academy Kids

| 维度 | 详情 |
|------|------|
| 目标年龄 | 2-8 岁 |
| 核心功能 | 计数、加减法、图形、模式识别、互动练习 |
| 玩法设计 | 个性化学习路径 + 互动游戏 + 即时反馈，彩色视觉风格 |
| 付费模式 | **完全免费**，无广告，无内购 |
| 优点 | 完整覆盖幼儿园阶段数学，无商业化干扰，对标 Common Core |
| 不足 | 游戏性偏弱，视觉吸引力一般，定制化选项有限 |
| App Store 评分 | 4.8 / 5 |

### 2.2 Moose Math (Duck Duck Moose)

| 维度 | 详情 |
|------|------|
| 目标年龄 | 3-7 岁 |
| 核心功能 | 数字识别、基础加减法、几何、问题解决 |
| 玩法设计 | **城市建造** — 完成数学任务获得建筑材料，逐步搭建虚拟城市 |
| 付费模式 | **完全免费** |
| 优点 | Stanford 教育学家参与设计，对标 Common Core K-1 标准，免费 |
| 不足 | 内容深度有限，长期留存不足 |
| 开发背景 | 被 Khan Academy 收购，技术已整合进 Khan Kids |

### 2.3 SplashLearn

| 维度 | 详情 |
|------|------|
| 目标年龄 | 2-11 岁 |
| 核心功能 | K-5 完整数学课程，AI 自适应难度 |
| 玩法设计 | 互动课程 + 测验 + 技能游戏 + 奖励/连胜机制 |
| 付费模式 | 免费试用 + 订阅 $7.99-$11.99/月 |
| 优点 | 4000 万+用户，课程对齐度高，AI 自适应系统成熟 |
| 不足 | 付费墙较深，免费内容极其有限 |
| App Store 评分 | 4.5 / 5 |

### 2.4 Todo Math

| 维度 | 详情 |
|------|------|
| 目标年龄 | 3-8 岁 |
| 核心功能 | 计数、加减法、时间、货币等概念 |
| 玩法设计 | 游戏化活动 + 每日挑战 + 进度追踪 |
| 付费模式 | 免费基础版 + 订阅 $9.99/月 或 $69.99/年 |
| 优点 | **无障碍设计出色** — 读写障碍友好字体、左手模式、包容性设计 |
| 不足 | 免费内容少，需订阅才能获得完整体验 |

### 2.5 DragonBox Numbers

| 维度 | 详情 |
|------|------|
| 目标年龄 | 4-8 岁 |
| 核心功能 | 数字概念、数感、早期代数思维 |
| 玩法设计 | **将数字具象化为可拆分/合并的角色**（Nooms），通过操纵角色理解数的分解与合成 |
| 付费模式 | 一次买断 $7.99-$24.99 |
| 优点 | 概念理解极深，教的不是"算术"而是"数学思维" |
| 不足 | 运算覆盖面窄，偏重概念探索而非运算练习 |

### 2.6 Marble Math Junior

| 维度 | 详情 |
|------|------|
| 目标年龄 | 5-8 岁 |
| 核心功能 | 计数、早期加法、问题解决 |
| 玩法设计 | **弹珠迷宫** — 通过倾斜/滑动控制弹珠，收集正确答案 |
| 付费模式 | 一次买断 $3.99 |
| 优点 | 触控体验好，3 级难度，16 种弹珠可定制 |
| 不足 | 内容量较少，覆盖面窄 |

### 2.7 Mathseeds

| 维度 | 详情 |
|------|------|
| 目标年龄 | 3-9 岁 |
| 核心功能 | K-3 数学结构化课程 |
| 玩法设计 | 奖励驱动的分级进阶系统，动画课程 + 测验 |
| 付费模式 | 订阅 $89/年 |
| 优点 | 结构化强，进阶清晰 |
| 不足 | 价格较高，缺乏创新玩法 |

### 2.8 Funexpected Math

| 维度 | 详情 |
|------|------|
| 目标年龄 | 3-7 岁 |
| 核心功能 | 10000+ 任务，涵盖算术、逻辑、编程思维、空间感 |
| 玩法设计 | 数字导师自适应引导 + 提示系统 |
| 付费模式 | 订阅 $10.99/月 或 $64.99/年 |
| 优点 | 多语言（16 种），获 Kidscreen Award 2025 |
| 不足 | 订阅费偏高 |
| App Store 评分 | 4.5 / 5 |

## 3. 国内市场主流产品

### 3.1 洪恩数学

| 维度 | 详情 |
|------|------|
| 目标年龄 | 3-12 岁 |
| 核心功能 | 数与量、加减计算、几何与空间、逻辑思维四大主题 |
| 内容规模 | 启蒙阶段 200+ 节正课，1000+ 精品互动 |
| 教学方法 | **"玩学练创"四步法** — 每个知识点拆分为 4 个环节循环巩固 |
| 特色 | "洪恩宇宙" IP 体系，中国风审美 |
| 优点 | 覆盖面广，品牌认知度高，内容体系完整 |
| 不足 | 游戏化过重 — 孩子易沉迷但学习效果不明显；题型设置单一；画面设计较混乱；不太适合零基础儿童 |

### 3.2 数感星球

| 维度 | 详情 |
|------|------|
| 目标年龄 | 3-10 岁 |
| 核心功能 | 数感能力培养 + 计算能力提升 |
| 内容规模 | 283 项数感技能，1500 个训练关卡，10000+ 次数感训练 |
| 教学方法 | **新加坡 CPA 教学法**（具象→图示→抽象），16 级别算术知识层级 |
| 特色 | 知识点覆盖学前至 4 年级，整数/分数/小数完整体系 |
| 优点 | 教学法专业，层级设计科学 |
| 不足 | 题目难度偏高，不适合启蒙阶段低龄幼童；部分游戏设计复杂，小孩子理解困难 |

### 3.3 都都数学 (Todo Math 国际版)

| 维度 | 详情 |
|------|------|
| 目标年龄 | 3-8 岁 |
| 核心功能 | 怪兽养成 + 每日数学任务 |
| 特色 | Apple Design Award 获奖，多语言支持 |
| 不足 | 国内版本更新滞后 |

## 4. 竞品玩法模式总结

| 玩法模式 | 代表产品 | 适用场景 |
|---------|---------|---------|
| 城市/农场建造 | Moose Math | 将答题奖励具象化为建造进度 |
| 角色/数字具象化 | DragonBox | 通过操纵"数字角色"理解概念 |
| 迷宫/弹珠物理 | Marble Math | 运动感知 + 数学结合 |
| 结构化课程闯关 | SplashLearn, Mathseeds | 系统化知识递进 |
| 每日任务 + 养成 | Todo Math, 洪恩 | 习惯养成 + 成就驱动 |
| AI 自适应 | SplashLearn, 数感星球 | 个性化难度调节 |

## 5. 市场空白与机会

### 5.1 供给侧缺口

1. **免费/低价 + 高质量的中文产品缺失** — Khan Academy Kids 模式在中文市场无对标产品
2. **iPad 原生体验被忽视** — 多数国内产品以手机为主，iPad 适配粗糙（仅放大而非重新设计）
3. **运算专项训练的游戏化不够极致** — 洪恩偏"广而浅"，数感星球偏"难而专业"
4. **亲子互动场景空白** — 大多数 App 仅提供后台家长报告，缺乏面对面亲子玩法

### 5.2 需求侧洞察

1. 5-6 岁儿童正处于幼小衔接关键期，20 以内加减法是刚需
2. 家长对"屏幕时间"有焦虑，需要"有限时间内高效学习"的产品
3. 国内教育政策强调素质教育、减负，纯刷题 App 不受欢迎
4. iPad 在家庭中的渗透率持续提升，大屏教育场景需求明确

### 5.3 差异化方向

- 聚焦 20 以内加减法 **做透做精**，不追求知识面广
- iPad 横屏原生设计，充分利用大屏触控和多点触控优势
- 故事驱动 + 实物化操作，让孩子理解"为什么"
- 亲子协作玩法作为核心差异点
